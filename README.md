# Geometria - Дизайн сайта в геометрическом стиле

Современный, адаптивный статический веб-сайт, демонстрирующий дизайн в геометрическом стиле. Этот проект показывает навыки во фронтенд-разработке, включая семантический HTML, SASS для стилизации и JavaScript для интерактивных элементов.

## Особенности

*   **Адаптивный дизайн:** Адаптируется к различным размерам экранов с помощью плавной типографики (`clamp()`) и медиа-запросов.
*   **Плавная навигация:** Улучшенный пользовательский опыт с плавными переходами к разделам страницы.
*   **Интерактивное бургер-меню:** Обеспечивает интуитивно понятную навигацию на мобильных устройствах.
*   **Современный CSS:** Использует SASS с переменными и миксинами для поддерживаемой и масштабируемой стилизации.
*   **Семантический HTML5:** Структурирован для лучшей доступности и SEO.

## Используемые технологии

*   HTML5
*   CSS3 (SASS)
*   JavaScript (ES6+)

## Настройка и разработка

Чтобы запустить локальную копию проекта, выполните следующие простые шаги.

### Предварительные требования

*   Веб-браузер (Chrome, Firefox, Safari и т.д.)
*   Node.js и npm (Node Package Manager) для компиляции SASS.

### Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/pvllsv/geo_dis
    cd YOUR_REPOSITORY_NAME/geo_dis
    ```
    

2.  **Установите SASS (если он у вас не установлен глобально):**
    ```bash
    npm install -g sass
    ```

3.  **Скомпилируйте SASS в CSS:**
    Перейдите в директорию `geo_dis` (или туда, где находится ваш `style.sass` относительно корня проекта).
    ```bash
    # Из корня проекта, предполагая, что 'geo_dis' содержит SASS-файлы
    sass geo_dis/sass/style.sass geo_dis/css/style.css
    ```
    Чтобы отслеживать изменения во время разработки:
    ```bash
    sass --watch geo_dis/sass/style.sass geo_dis/css/style.css
    ```

4.  **Откройте веб-сайт:**
    Перейдите в директорию `geo_dis` и откройте `index.html` в вашем веб-браузере.

    ```bash
    # Из корня проекта
    start geo_dis/index.html # Для Windows
    open geo_dis/index.html # Для macOS
    xdg-open geo_dis/index.html # Для Linux
    ```

## Использование

Просто откройте `geo_dis/index.html` в вашем браузере, чтобы просмотреть веб-сайт. Перемещайтесь по разделам с помощью меню в шапке или бургер-меню на мобильных устройствах.
